# Code of Conduct

Be respectful, specific, and constructive.

This project deals with document conversion edge cases, and many bug reports will involve partial failures. Keep discussion focused on reproducible behavior, test data that can be legally shared, and practical fixes.

Unacceptable behavior includes harassment, personal attacks, deliberate disruption, and sharing private or copyrighted documents without permission.

Project maintainers may remove comments, close issues, or block users when needed to keep the project safe and useful.
